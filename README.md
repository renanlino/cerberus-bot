# Pump & Dump Bot Assistant for Cryptopia #

Bot semi-automático de operação Pump&Dump com suporte Multithread para a exchange Cryptopia

## Features ##
- Configuração de margem de pump inicial
- Configuração de meta de ganho percentual
- Capacidade de operação em múltiplas moedas
- Suporte completo à múltiplas threads para otimizar desempenho
- Suporte à sessão HTTP para diminuição de latência
